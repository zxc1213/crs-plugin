# /req-conv:view

查看对话记录的详细内容。

## 用法

```
/req-conv:view <对话ID> [选项]
```

## 选项

| 简写 | 完整       | 默认值 | 描述                                      |
| ---- | ---------- | ------ | ----------------------------------------- |
| `-f` | `--format` | pretty | 输出格式：pretty、json、markdown、compact |

## 示例

```bash
# 查看对话详情（美化格式）
/req-conv:view 20260522-001

# JSON 格式输出
/req-conv:view 20260522-001 --format json

# Markdown 格式输出
/req-conv:view 20260522-001 -f markdown

# 紧凑格式输出
/req-conv:view 20260522-001 -f compact
```

## 输出格式

### Pretty 格式（默认）

```
═══════════════════════════════════════════════════════════════
  对话 ID: 20260522-001
  开始时间: 2026-05-22 14:30:00
  状态: active
  标签: #feature #api
═══════════════════════════════════════════════════════════════

[14:30:15] 👤 用户:
实现用户登录功能

[14:30:20] 🤖 助手:
好的，我来实现用户登录功能...

[14:35:10] 🔧 工具调用: Edit
  文件: src/auth/login.js
  修改: +42 行

═══════════════════════════════════════════════════════════════
```

### Compact 格式

```
20260522-001 | 2026-05-22 14:30 | active | 42 消息
```

### JSON 格式

完整的 JSON 数据结构，包含所有字段。

## 相关命令

- `/req-conv:list` - 列出所有对话
- `/req-conv:search` - 搜索对话
- `/req-conv:mark` - 标记对话
