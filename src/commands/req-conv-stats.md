# /req-conv:stats

显示对话记录的统计信息。

## 用法

```
/req-conv:stats [选项]
```

## 选项

| 简写 | 完整       | 默认值 | 描述                            |
| ---- | ---------- | ------ | ------------------------------- |
| `-f` | `--format` | table  | 输出格式：table、json、markdown |

## 示例

```bash
# 显示统计信息（表格格式）
/req-conv:stats

# JSON 格式输出
/req-conv:stats --format json

# Markdown 格式输出
/req-conv:stats -f markdown
```

## 统计指标

| 指标       | 说明                   |
| ---------- | ---------------------- |
| 总对话数   | 所有对话记录总数       |
| 总消息数   | 所有对话的消息总数     |
| 总工具调用 | 所有对话的工具调用总数 |
| 平均消息数 | 每个对话的平均消息数   |
| 平均评分   | 所有已评分对话的平均分 |

### 状态分布

按状态（active/archived/important）统计对话数量。

### 标签使用排行

使用次数最多的前 10 个标签。

### 最近活动

最近创建的 5 个对话记录。
